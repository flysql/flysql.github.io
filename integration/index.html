System Context diagram

Git Repository

  • Die im Repository hinterlegten Dateien werden als REST Resource zur Verfügung gestellt

Testausführungssoftware

  • Beliebige Applikationen, die das REST Protokoll beherrschen, können mit FlySql integriert werden
  • Kommerzielle Tools zur Testautomatisierung wie zum Beispiel Tricentis Tosca, Ranorex, Service Trace
  • Integration in bestehende End-to-end Tests, zum Beispiel E-Banking
  • Open Source Software zur Integration in bestehende End-to-end Tests, z.B. Selenium, Cypress
  • Skalierbares Regressionstesten von Avaloq ACP mit zB Java Junit

Kubernetis Platform

  • Stellt die Konfiguration (Datenbank Verbindung, Git Repository Url, Zertifikate) zur Verfügung
  • Skalierung und Betrieb

Avaloq Core System

  • FlySql wird SQL Skripts, PL/SQL Statements und Avaloq Script Statements hier zur Ausführung bringen

Container diagram

FlySql - Repository

  • Aktualisiert das Git Repository, welches durch Certified Avaloq Professional gepflegt wird
  • Stellt die Verbindung von REST Resource zu den SQL Statements, PL/SQL Scripte und Avaloq Script Statements her
  • Sorgt für die Auflösung der Parameter falls solche in den Skripten und Statments vorhanden sind

FlySql - Executor

  • Kann die Verbindung (jdbc) zu einer Datenbank aufbauen
  • Führt die SQL Statements, PL/SQL Scripte und Avaloq Script Statements aus
  • Bereitet Ergebnisdaten der SQL Statements, PL/SQL Scripte und Avaloq Script Statements für die REST Response auf

FlySql - Migrator

  • Unterstützt die Migration von bestehenden AST Tests in die FlySql Welt
  • Liest aus einem AST Test Skript Repository die notwendigen SQL Statements, PL/SQL Scripte und Avaloq Script Statement aus
  • Erzeugt Java Junit Code für die meisten AST Befehle, mit denen sofort gearbeitet werden kann
